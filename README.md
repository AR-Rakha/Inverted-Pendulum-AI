# Inverted Pendulum AI

## Download
Download the SIM folder and dqn_pendulum.pth

## Required Python Libraries
[pygame, numpy, torch]
pip install torch
pip install numpy
pip install pygame


## Run Program
Open the folder with SIM and dqn_pendulum.pth in VSCode
run InvertedPendulum_SIM.py

## Controls

### reset
keys: [r]

### Control player pendulum
keys: [arrow left, arrow right]

### Add torque to AI pendulum
keys: [a,q]

## Target
Balance the pendulum
